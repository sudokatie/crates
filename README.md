# Crates

Browser-based Sokoban. Push crates onto targets. Pure puzzle.

## Play

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Controls

- Arrow keys or WASD: Move
- U: Undo
- R: Restart level
- N/P: Next/Previous level

## Rules

1. Push crates onto green target spots
2. Can't pull crates
3. Can't push two crates at once
4. All targets covered = level complete

## Features

- 20 Microban levels (beginner-friendly)
- Unlimited undo
- Progress saved automatically
- Best solution tracking

## Tech Stack

- Next.js 14 + TypeScript
- HTML5 Canvas
- Tailwind CSS
